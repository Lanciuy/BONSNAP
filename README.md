## 🌟 What is BONSNAP? (The Tea ☕)

Forget aplikasi finance yang kaku dan boomer banget. **BONSNAP** is here to deliver that *main character energy* buat dompet kalian. Ini adalah Expense Tracker next-gen yang didesain khusus biar tracking pengeluaran kerasa *effortless, premium,* dan tentunya sangat *expensive*. 

Mau split bill habis nongkrong ngopi cantik, atau budgeting buat checkout keranjang e-commerce? BONSNAP bakal keep your finances in check. It's not just an app, it's a lifestyle, bestie. 💅✨

## 🔥 Core Flex (Fitur Utama)

- **💅 Immaculate UI/UX**: Tampilannya *mahal* banget (biar uang kalian yang terselamatkan). Dipenuhi dengan slick micro-animations dan transisi yang buttery smooth.
- **📊 Aesthetic Analytics**: Visualisasi spending habits dengan chart yang *gorgeous* dan gampang dibaca, nggak bikin overthinking.
- **💸 Effortless Tracking**: Log pengeluaran kalian lebih cepet dari pada bilang "retail therapy".
- **🔒 Secure & Private**: Dompetmu, rahasiamu. No drama.

## 🚀 How to Run (Start the Vibe)

Gak usah overthinking, tinggal run aja:

```bash
# Install the goods
npm i

# Start the dev server
npm run dev
```
